Draw a blade

To get you started, use the beginShape() method below to create one of the blades in the p5 editor. For more info on this method, see the p5.js reference

function setup() {
  createCanvas(400, 400);
}

function draw() {
	background(220);
	// Copy and paste this into the code editor below
	// change the vertexes to get your blade
	//looking like the example
	beginShape(QUADS);
	vertex(0, 0);
	vertex(7, 20);
	vertex(40, 20);
	vertex(47, 0);
	endShape();
}
